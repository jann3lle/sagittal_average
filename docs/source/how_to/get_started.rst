Getting Started
===============

Installing the Library
----------------------

To install this package locally:

.. code-block:: bash

    pip install -e .

This installs the command-line tool:

.. code-block:: bash

    sagittal_average_run

You can then run the program:

.. code-block:: bash

    sagittal_average_run brain_sample.csv -o brain_average.csv

Now you're ready to use the library...
